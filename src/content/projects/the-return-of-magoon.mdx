## Overview

The Return of Magoon is a small Unity 2D action-adventure demo inspired by Zelda-like exploration and combat. The project was developed as part of a final-year course and involved a team of five programmers working in parallel.

## My contribution

- Worked on gameplay features for combat, exploration and map interaction.
- Supported item usage, puzzle interactions and enemy encounters.
- Helped build a complete demo loop with multiple enemy types, a final boss and dialogue.

## Technical focus

This project is useful as a gameplay breadth example: it combines combat, exploration, progression and level interaction in a compact student production.
