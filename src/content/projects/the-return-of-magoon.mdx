## Overview

The Return of Magoon is a compact C# 2D action-adventure demo inspired by Zelda-like exploration and combat. It is intentionally presented as a smaller project, useful for showing breadth across combat, item usage, dialogue and puzzle interactions.

## My contribution

- Worked on gameplay features for combat, exploration and map interaction.
- Supported item usage, puzzle interactions and enemy encounters.
- Helped build a complete demo loop with multiple enemy types, a final boss and dialogue.

## Technical focus

This project is a breadth example rather than a headline case study: it combines combat, exploration, progression and level interaction in a compact student production.
